# NonParametric_Clustering
Categorizing countries worldwide using socio-economic and health data through non-parametric clustering

# ■ Non-Parametric Bayesian Clustering of Countries

This project explores **non-parametric Bayesian clustering** methods to categorize countries worldwide based on socio-economic and health indicators. It is part of a methodological study on advanced clustering techniques, followed by a practical application to real-world country data.

---

## ■ Introduction

The project focuses on **Bayesian non-parametric clustering**, which allows flexible grouping without pre-specifying the number of clusters. Key concepts include **loss functions** such as **Binder loss** and **Variation of Information (VI)**, which help evaluate clustering performance.

---

## ■ Methods

– Implementation of **Greedy algorithms** and the **SALSO algorithm** for clustering optimization.  
– Simulation studies to assess the performance and robustness of the clustering approaches.  
– Application of the methods to a dataset of countries, using socio-economic and health indicators to identify meaningful clusters.

---

## ■ Data

The applied dataset includes a range of **socio-economic and health indicators** for countries worldwide.  
> *Note: The dataset itself is not included due to licensing restrictions, but it can be accessed from public sources.*

---

## ■ Results

– Identification of clusters of countries with similar socio-economic and health profiles.  
– Comparative analysis of algorithm performance using **Binder loss** and **VI** metrics.  
– Visualization of cluster assignments for clear interpretation of patterns.
